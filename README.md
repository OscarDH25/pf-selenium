\# pf-selenium



QA portfolio project focused on testability and automation.

\- Fullstack app designed for testability (data-testid, seed/reset)

\- E2E automation framework in Java (Selenium 4, JUnit 5)

\- CI pipeline with reports and artifacts



\## Project structure

\- /app      -> backend + frontend

\- /e2e      -> selenium automation framework
